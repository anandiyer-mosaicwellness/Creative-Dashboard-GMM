How to Refresh This Dashboard Weekly
  1. Go to Google Ads → Reports → Predefined Reports → Other → Asset Association Report
  2. Set the date range to the new week and download as .xlsx
  3. Place the new file in the uploads folder alongside the existing files
  4. Run the Python refresh script:
    python generate_dashboard.py
  5. Open the newly generated MM_YT_Dashboard.html in your browser — done!
💡 Tip: All previous weekly files must remain in the uploads folder. The script reads all of them together to maintain the full historical ranking view. The dashboard is a single self-contained HTML file — share it with your team by email or Slack.
